Software Freedom Day

Join us on Software Freedom Day (SFD), organized by Computer science department with support from Computer science department, to celebrate and promote Free and Open Source Software (FOSS) in C through exciting talks, panel discussion, and networking opportunities!

Free And Open Software

Free and Open Source Software (FOSS) refers to software that anyone is freely licensed to use, copy, study and change in any way that they want, and the source code is openly shared so that people are encouraged to volunterily improve the design of the software. Use of free software encourages learning and sharing what we have learned with others in the community. Other benifits of using FOSS would be to decreased software cost, protecting privecy and providing more control to the users.
